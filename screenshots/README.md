# Screenshots

Bu klasöre dashboard ve sistem ekran görüntülerini ekleyin.

Önerilen dosyalar:
- `dashboard.png` - Ana dashboard görünümü
- `comparison.png` - Fiyat karşılaştırma tablosu
- `charts.png` - Grafikler ve analizler
- `features.gif` - Animasyonlu demo (opsiyonel)

## Nasıl Eklerim?

1. Streamlit dashboard'u çalıştırın
2. Ekran görüntüsü alın (Windows: Win+Shift+S, Mac: Cmd+Shift+4)
3. Bu klasöre kaydedin
4. README.md'de kullanın:

```markdown
![Dashboard](screenshots/dashboard.png)
```
